Storage Jars / Cache Stash Web Service

Finds all cache directories and creates web pages showing all requests, and enabling you to view cached files through a browser. Runs a web server to enable browsing.

To run the Storage Jars / Cache Stash Web Service:

  1. Open a command shell
  2. Navigate to [Install_directory]/final_project
  3. Execute python storage_jars.py
  4. Open a browser
  5. In the browser, access http://localhost:56565/
  6. From Cache Contents Web Service, browse to:
    • Storage Jars is a page listing _CACHE_ file contents, by absolute filepath of each _CACHE_ file. For each request, it lists:
      • key: link to the online resource
      • fetch count: number of times the request was made
      • last fetched: date and time of last request
      • last modified: date and time of last modification
      • data size: memory used by request data
    • Cache Stash is a list of links to unzipped versions of cached files, so you can see what they are.
    • Help takes you to this help page.

Surf Spy Client-Server

The server monitors the first _CACHE_001_ file it finds, and reports on every new url request. The client allows you to remotely monitor which websites are being requested - so'ze you'ze can spy on people, see?!

The server has a generator which trails the first _CACHE_001_ file it finds, filtering out all but the domain name, which it sends over a socket to a remote client. The client listens on the socket, and simply reports any domains sent from the server.

To run Surf Spy Client-Server:

  1. Open a command shell
  2. Navigate to [Install_directory]/final_project
  3. Execute python surf_spy_server.py
  4. Open a second command shell
  5. Navigate to [Install_directory]/final_project
  6. Execute python surf_spy_client.py
  7. Open a browser and surf, to see it work
